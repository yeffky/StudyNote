# 约束

## 一、非空约束（not null）

当发生错误时不会占用auto_increment的自增值

## 二、唯一约束（唯一键）（unique）

当发生错误时会占用auto_increment的自增值

## 三、主键约束（primary key）

## 四、默认约束（default）

默认值

## 五、检查约束（check）

检查是否符合约束条件

## 六、外键约束 （foreign key）

用于让两张表的数据之间建立连接，从而保证数据的一致性和完整性

### 1. 添加外键

```mysql
alter table 表名 add constraint 外键名称 foreign key（外键字段名） references 主表(主表列名);
```

### 2. 删除外键

```mysql
alter 表名 drop foreign key 外键名称;
```

### 3. 外键删除/更新行为

* NO ACTION：有则不允许操作
* RESTRICT：同NO ACTION
* CASCADE：级联操作
* SET NULL：删除时设置外键值为null
* SET DEFAULT：子表将外键列设置为默认值（innodb不支持）

```mysql
alter table 表名 add constraint 外键名称 foreign key（外键字段名） references 主表(主表列名) on update set null on delete set null;
```
