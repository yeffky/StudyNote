# 函数

## 一、字符串函数

### 1.左右填充

```mysql
lpad & rpad(str, num, char);
str：字符串
num：目标长度
char：填充字符
```

应用场景：格式化数据库数据

### 2. 提取子串

```mysql
substring(str, start, end);
```

## 二、数值函数

### 1. 取整函数

```mysql
ceil(x)：向上取整
floor(x)：向下取整
```

### 2. 四舍五入

```mysql
round(x, y)：x四舍五入，保留y位
```

### 3. 随机数

```mysql
rand()：生成0-1内的随机数
```

应用场景：生成固定长度随机数

## 三、日期函数

### 1.当前时间

```mysql
curdate()：日期
curtime()：当前时间
```

### 2.时间间隔

```mysql
datediff(date1, date2)
```

应用场景：求天数

## 四、流程函数

### 1. if函数

```mysql
if(value, t, f)：如果value为true返回t，否则f
ifnull(value, t, f)：如果value不为空...
```

### 2. case函数

```mysql
case when then else end
```
