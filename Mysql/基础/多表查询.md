# 多表查询

判断表和连接条件形成语句

## 一、多表关系

### 1. 一对多

* 案例：部门和员工的关系
* 关系：一个部门对应多个员工，一个员工对应一个部门
* 实现：在多的一方建立外键，指向一的一方的主键

### 2. 多对多

* 案例：学生与课程的关系
* 关系：一个学生选修多门课程，一门课程可以供多个学生选择
* 实现：建立第三张中间表，包含两个外键，关联双方主键

### 3. 一对一

* 案例：用户和用户详情
* 关系：用于单表差分，将一张表的基础字段放在一张表，其他详情字段放在另一张表
* 实现：在一方加入外键，关联另一方的主键，设置外键为唯一的unique

## 二、消除笛卡尔积

```mysql
select * from t where 字段1 = 字段2
```

## 三、多表查询分类

### 1. 连接查询

* 内连接：查询A、B交集部分数据
* 外连接
  * 左外连接：查询左表所有数据以及交集数据
  * 右外连接：查询右表所有数据以及交集数据
* 自连接：与自身连接查询，自连接必须使用表别名

### 2. 子查询

## 四、内连接

### 1. 隐式内连接

``` mysql
select fieldlist from t1, t2 where condition;
```

### 2. 显式内连接（效率更高）

```mysql
select fieldlist from t1 join t2 on condition;
```

## 五、外连接

### 1.左&右外连接

```mysql
select * from t1 left outer join t2 on condition;
```

## 六、自连接

```mysql
select fieldlist from t1 name1 join t1 name2 on condition;
```

## 八、子查询（嵌套查询）

```mysql
select * from t1 where column1 = (select column1 from t2);
```

### 1. 标量子查询（效率优于内连接）

子查询结果为单个值

### 2. 列子查询

子查询返回结果是一列

常用操作符：IN、NOT IN、ANY、SOME、ALL

```mysql
select * from t1 where column1 >&=&< 操作符(select column1 from t2);
```

### 3. 行子查询

子查询结果返回一行多列

```mysql
select * from t1 where (column1, column2) = (select column1, column2 from t2 where condition);
```

### 4. 表子查询

子查询返回结果为多行多列

```mysql
select * from t1 where (column1, column2) 操作符 (select column1, column2 from t2 where condition);
```

```mysql
select e.*, d.* from (select * from t1 WHERE condition) e LEFT JOIN t2 d on condition;
```

## 复杂的多表查询

统计各部门员工数量：

```mysql
select d.id, d.name, (SELECT COUNT(*) from emp e WHERE e.dept_id = d.id) '人数' from dept d;
```

统计各部门薪资低于平均薪资员工：

```mysql
SELECT * from emp e2 WHERE e2.salary < (select avg(salary) from emp e1 WHERE e1.dept_id = e2.dept_id);
```
