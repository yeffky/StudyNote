# 存储过程

* 实现经过编译并存储在数据库中的一段SQL语句的集合，调用存储过程减少数据在数据库和服务器之间的传输
* 是SQL语言层面的代码封装和重用，可以接收参数和返回数据

## 一、存储过程操作

### 1.创建

```mysql
create procedure procedure_name(parameter)
begin
    SQL
end;
```

### 2. 调用

```mysql
call procedure_name (parameter);
```

### 3. 查看

```mysql
# 查询指定数据库的存储过程和状态信息
select * from information_schema.routines where routine_schema='xxx';
# 查询某个存储过程的定义
show create procedure procedure_name;
```

### 4. 删除

```mysql
drop procedure procedure_name;
```

## 二、变量

### 1. 系统变量

* 查看

```mysql
show [session|global] variables;
show [session|global] variables like '...';
select @@[session|global] 系统变量名;
```

* 设置

```mysql
set [session|global] variable_name = ...;
set @@[session|global] variable_name = ...;
```

注：重启后变量恢复为默认状态（初始值），要改系统配置文件才能修改初始值

### 2. 用户定义变量

根据需要定义的变量，不用提前声明，使用时直接用"@variable_name"，作用域为当前连接

* 赋值

```mysql
set @var_name = expr[, @var_name=expr]...;
set @var_name := expr[, @var_name:=expr]...;
select @var_name:= expr[, @var_name:=expr]...;
select 字段名 into @var_name from table_name;
```

* 使用

```mysql
select @var_name;
```

### 3. 局部变量

局部变量需要declare声明（没有@），可以在存储过程内部使用或作为输入参数

* 声明

```mysql
declare var_name var_type [default]
```

* 赋值

```mysql
set var_name = ...;
set var_name := ...;
select field into var_name from table_name;
```

## 八、loop

* leave：退出循环
* iterate：必须用在循环中，作用是跳过当前循环剩下的语句，进入下一次循环

```mysql
[begin_label:] loop
    SQL
end loop[end_label];

# 退出循环
leave label;
# 进入下一次循环
iterate label;
```

## 九、cursor

cursor用于存储查询结果集的数据类型，游标的使用包括游标声明、open、fetch和close

### 1. 声明游标

```mysql
declare cursor_name cursor for SQL;
```

### 2. 打开游标

```mysql
open cursor;
```

### 3. 获取游标记录

```mysql
fetch cursor_name into var_name[, var_name];
```

### 4. 关闭游标

```mysql
close cursor;
```

### 5. 举例

根据传入参数uage查询tb_user中所有年龄小于uage的name和profession，并插入新表

```mysql
CREATE DEFINER=`root`@`%` PROCEDURE `p11`(in uage int)
begin
 DECLARE u_cursor CURSOR for select name, profession from tb_user WHERE age <= uage;
 DECLARE uname VARCHAR(100);
 DECLARE upro VARCHAR(100);
 drop table if EXISTS tb_user_pro;
 create table if not EXISTS tb_user_pro(
  id int PRIMARY key auto_increment, 
  name varchar(100), 
  profession VARCHAR(100)
 );
 
 open u_cursor;
 while true do
  fetch u_cursor into uname, upro;
  insert into tb_user_pro VALUES(null, uname, upro);
 end while;
 close u_cursor;
end
```

## 十、条件处理程序

用于定义在流程控制结构执行过程中遇到问题时相应的处理步骤

```mysql
declare handler_action handler for condition_value [,condition_value]... statement;

handler_action
    continue;
    exit;
condition_value
    SQLSTATE sqlstate_value：状态码
    SQLWARNING:所有以01开头的SQLSTATE的缩写
    NOT FOUND:所有以02开头的SQLSTATE的缩写
    SQLEXCEPTION：除了上述两种情况的其他情况
```

## 十一、存储函数

```mysql
create function func_name(para_name)
returns type [characteristic...]
begin
    SQL
    return ...;
end;
```

characteristic：

* deterministic：相同的输入参数总是产生相同的结果
* no sql：不包含sql
* reads sql data：包含读取数据的语句，不包含写入数据的语句
