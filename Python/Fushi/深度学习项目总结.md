# 项目总结

## 一、MData类

### 1、__init__

包含两个路径，分别负责导入图片和标签

### 2、__getitem__

负责将导入路径进行处理并对每一张图片进行transform处理，并将其转换为tensor张量

### 3、transform函数

使用PIL库的Image模块对图片进行处理

#### （1）创建新的RGB图像，大小为224*224，背景为白色，使用ColorJitter随机调整图像参数

## 二、Loader类

### 1、def __init__(self, dataset, test_persent)

划分训练集和测试集

### 2、 def getDataLoader(self, batch_size, num_workers, shuffle=True)

接受三个参数：batch_size 表示每个批次的样本数量，num_workers 表示用于数据加载的线程数量，shuffle 表示是否在每个 epoch 重新打乱数据。

## 三、训练流程

### 1、指定设备

定义了训练的设备为 CUDA（如果可用的话），并将模型 net 移动到该设备上。

### 2、定义

指定了训练的总轮数 NUM_EPOCHS，初始化了一个初始损失值 best_loss，并定义了优化器 opt 和损失函数 loss_fun。同时指定了保存最佳模型权重的路径 BEST_MODEL_PATH。

### 3、循环训练

    在每个 epoch 的训练循环中将 train_loss 初始化为 0
    遍历 train_loader 中的每个批次，每次都执行以下操作：
    将优化器的梯度清零。
    将输入图像和分割标签移动到指定的设备上。
    通过模型 net 对输入图像进行前向传播，得到输出。
    计算输出与分割标签之间的损失。
    将损失值累加到 train_loss 上。
    执行反向传播和优化器的一步参数更新。
    计算每个 epoch 结束后的平均训练损失。
    在每个 epoch 结束后，进入测试阶段：

### 5、测试

    将 test_loss 初始化为 0
    遍历 test_loader 中的每个批次，每次都执行以下操作：
    将输入图像和分割标签移动到指定的设备上。
    通过模型 net 对输入图像进行前向传播，得到输出。
    计算输出与分割标签之间的损失。
    将损失值累加到 test_loss 上。
    计算每个 epoch 结束后的平均测试损失。
    打印每个 epoch 结束后的训练损失和测试损失。

### 6、更新参数

如果当前 epoch 的训练损失 train_loss 小于之前记录的最佳损失值 best_loss，则保存当前模型的参数为最佳模型参数，并更新 best_loss。
