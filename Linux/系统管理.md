# 进程管理

## 一、查看进程

### 1. ps（进程操作）

```shell
ps [选项]
选项：
    a：显示一个终端所有进程，除了会话引线
    u：显示进程的归属用户和内存使用情况
    x：显示没有控制终端的进程
    -l：长格式显示
    -e：显示所有进程
    -f：把进程的所有信息都显示出来
常用组合：
    ps aux
    ps -ef | grep
```

## 二、杀死进程

### 1. kill

```shell
kill [signal] PID
信号：
    -1：让主进程重启动
    -19：让进程暂停
```

## 三、文件操作

### 1. lsof（文件操作）

```shell
lsof [选项]
选项：
    -u：列出指定用户已经打开的文件，选项后接用户名
    -i：列出所有打开的网络文件，加：端口号可以指定端口
```
