# 常用命令

## 一、目录操作

### 1. ls

```shell
ls -l(ll)：长格式显示
权限-引用计数-所有者-所属组-大小-修改时间-文件名

```

### 2. mkdir & rmdir

```shell
mkdir -p：递归创建
rm -rf dir：删除目录下所有文件和目录
-f：只读也删除，无需确认
-r：目录和目录下文件注意删除
```

---

## 二、文件操作

### 1. touch

创建文件或修改时间戳

### 2. stat

查看文件详细信息

### 3. ln

```shell
    ln -s：创建软链接
```

### 4. 权限位含义

例：-rw-r--r--.1

- 第1位：文件类型

&emsp;&emsp;'-'：普通文件
&emsp;&emsp;'b'：块设备文件（存储设备）
&emsp;&emsp;'d'：目录文件
&emsp;&emsp;'l'：软链接文件（鼠标键盘）
&emsp;&emsp;'s'：套接字文件

- 第2-4，5-7，8-10位：
&emsp;&emsp;分别代表所有者、组、其他人权限

### 5. chmod

修改权限
&emsp;&emsp;-R：递归设置权限

```shell
chmod 权限模式 文件名
常用权限：
    1、644：文件基本权限。所有者：读、写； 所属组和其他人：只读。
    2、755：文件执行权限和目录基本权限。所有者：读、写、执行； 所属组和其他人：读、执行。
    3、777：最大权限。
```

### 6. chown

修改文件和目录所有者和所属组

## 四、压缩解压命令

### 1. zip&unzip

```shell
zip [选项] 压缩包名 源文件或目录
选项：
    -r：压缩目录
upzip [选项] 压缩包名
选项：
    -d：指定解压位置
```

### 2. tar

```shell
打包：tar [选项] [-f 压缩包名] 源文件或目录
选项：
    -c：打包
    -f：指定文件名
    -v：显示过程
    -z:.tar.gz格式
    -j：.tar.bz2格式

解打包：tar [选项] 压缩包名
选项：
    -x：解打包
    -f：指定文件名
    -v：显示过程
    -t：测试，不解打包
    -C：指定打包位置
常用：
    tar -zxvf [压缩包名] -C [目录] [压缩包内特定文件名]
```

## 五、常用网络命令

### 1. ifconfig

主要用于查看IP地址信息，inet即IP地址（ipv4）

### 2. ping

```shell
ping [选项] IP
选项：
    -b：IP要指定广播地址，对网段进行探测
    -c + 次数：指定ping的次数
    -s + 字节：指定探测包大小
```

### 3. netstat

### 4. traceroute

```shell
traceroute [选项] IP
选项：
    -I：使用ICMP协议
    -U：使用UDP协议
    -T：使用TCP协议
    -p：指定端口
    -n: 使用IP地址，不使用域名
常用：
    traceroute -T -n -p 88 xxx.xxx.xxx.xxx
```

### 5. nslookup

```shell
nslookup [选项] 域名
选项：
    -q：指定查询类型

```shell
netstat [选项]
选项：
    -a：列出所有网络状态
    -c + 秒数：指定每隔几秒刷新一次
    -n：使用IP和端口号显示
    -p：显示PID和程序名
    -t&-u：显示TCP&UDP端口连接
    -l：显示LISTEN状态连接
    -r：显示路由表
```

## 七、实战补充

### 1.查看某机器端口是否开放

```shell
telnet A.IP A.port
```

### 2. nohup 命令

```shell
nohup ./test.sh > output.log 2>&1 &
说明：
">"符号用于将命令的`标准输出`重定向到指定的文件中
> outlog.log 表示将终端输出保存到outlog.log文件中
"2>"表示将命令的`标准错误`输出重定向到指定的文件中
2>&1 表示将命令的标准错误输出重定向到标准输出1中一起输出 （ps:标准输出1指的是>outlog.log文件，具体可见下面的内容）
& 后台挂起
```
