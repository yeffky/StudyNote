# Vue组件化开发

## 一、NPM使用

* NPM（Node Package Manager）：NodeJS包管理和分发工具
* NPM：安装和更新依赖（类似于Maven和conda）
* package.json类似于pom.xml

## 二、VueCli

* VueCli脚手架，用于快速搭建带有热重载（LiveServer）及构建生产版本等功能的单界面应用
* 基于webpack，页可以通过项目的配置文件进行配置

### 1.创建vue项目

* 安装 npm install -g @vue/cli
* 对应文件夹vue ui可视化创建项目

## 三、组件构成

* template：组件的魔板结构，可以包含HTML标签和其他组件
* script：组件的JS
* style：样式

## 四、组件使用

* 导入并注册组件
* 可以在组件中定义props请求APP.vue进行参数传递

```Vue
// App.vue 
<template>
  <div id="app">
    <!-- <img alt="Vue logo" src="./assets/logo.png"> -->
    <!-- <HelloWorld msg="Welcome to Your Vue.js App"/> -->
    <Movie v-for="movie in movies" :key="movie.id" :rating="movie.rating" :title="movie.title" />
  </div>
</template>

<script>
//局部注册
import HelloWorld from './components/HelloWorld.vue'
import Movie from './components/Movie.vue'
export default {
  name: 'App',
  components: {
    HelloWorld, 
    Movie
  }, 
  data() {
    return {
      movies: [
        {"id": 1, title: "金刚狼1", rating: "8.8"},
        {"id": 2, title: "金刚狼2", rating: "8.8"},
        {"id": 3, title: "金刚狼3", rating: "8.8"}
      ]
    }
  },
}
</script>

// Movie.vue
<template>
    <div>
        <h1>{{title}}</h1>
        <span>{{rating}}</span>
        <button @click="fn">点击收藏</button>
    </div>
</template>

<script>
export default {
    name: 'Movie',
    props:["title", "rating"],
    data() {
        return {
        };
    },
};
</script>
```

## 五、全局注册

ElementUI在多处使用，进行全局注册

```JS
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
Vue.config.productionTip = false
//全局注册
Vue.use(ElementUI);
```
